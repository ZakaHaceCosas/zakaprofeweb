# ZakaProfe Web

Web/app de [ZakaProfe](https://youtube.com/@ZakaProfe) :]

Contiene enlace al canal, enlace de contacto (mas un horario básico para saber que esperar de mi respuesta), un buscador (más rápido que el propio de YouTube, ja), una calculadora de notas (porque sí), y poco más. Tengo que pensar más ideas, ya se me ocurrirán.
